# ゴンダ様からの車両騒音苦情対応プロジェクト

このリポジトリは、岡崎車輌部品株式会社の近隣住民（ゴンダ様）からの車両騒音苦情に対応するための専用プロジェクトリポジトリです。

## プロジェクト概要

- **課題**: 会社事務所の隣にお住まいのゴンダ様からの車両騒音苦情への対応
- **目的**: 詳細確認、社内検討、改善策立案、実施、フォローアップ
- **難易度**: 小（1-2週間）
- **優先度**: medium
- **ビジネスインパクト**: 近隣住民との良好な関係維持は事業継続の基盤

## リポジトリ構成

```
.
├── README.md              # このファイル
├── CLAUDE.md              # Claude Code ガイダンス
├── .obsidian/             # Obsidian Vault 設定
├── 00_inbox/              # 新規タスク・問題の一時格納
├── 01_research/           # 詳細ヒアリング・情報収集
├── 02_planning/           # 対応方針の検討・計画
├── 03_implementation/     # 改善策の実装
└── 04_followup/           # フォローアップ・検証

```

## プロジェクト進捗

- [ ] **Phase 1: 詳細ヒアリング** - ゴンダ様からの苦情内容を詳しく確認
- [ ] **Phase 2: 社内検討** - 対応方針を決定、経営層の承認取得
- [ ] **Phase 3: 改善策の実施** - 決定した対応内容を実行
- [ ] **Phase 4: フォローアップ** - ゴンダ様への報告、改善状況確認
