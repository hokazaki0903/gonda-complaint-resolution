## 本プロジェクトでの進行手順

このリポジトリでは、以下のフェーズで対応を進めます：

### Phase 1: 詳細ヒアリング（01_research/）

ゴンダ様からの苦情内容を詳しく確認します。

- [ ] ゴンダ様への連絡・アポイント取得
- [ ] ヒアリング実施（時間帯、頻度、影響度、具体的な改善要望など）
- [ ] 現在の運行状況の確認
- [ ] 騒音規制法など法的背景の調査

### Phase 2: 社内検討（02_planning/）

社内で対応方針を検討・決定します。

- [ ] 社内会議の開催（経営層、営業責任者、運行管理担当者）
- [ ] 複数の改善案の検討（ルート変更、時間帯調整など）
- [ ] 実行可能な対応策の絞り込み
- [ ] 実施スケジュール・責任者の決定
- [ ] 経営層の承認取得

### Phase 3: 改善実施（03_implementation/）

決定した対応内容を実装します。

- [ ] 運行ルート・スケジュール変更の実施
- [ ] 関連部署への指示・通知
- [ ] 実施状況の記録

### Phase 4: フォローアップ（04_followup/）

ゴンダ様への報告と効果測定を行います。

- [ ] ゴンダ様への報告・説明
- [ ] 改善状況の定期確認（1ヶ月後、3ヶ月後など）
- [ ] 満足度の確認
- [ ] 必要に応じた追加対応の検討
